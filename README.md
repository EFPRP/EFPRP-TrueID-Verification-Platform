# EFPRP TrueID Verification Portal

**Official Identity Authentication Infrastructure for the European Fire Prevention & Rural Preservation Organization.**

## Overview
This repository hosts the front-end architecture for the **TrueID Verification System** (accessible via [verify.efprp.org](https://verify.efprp.org)). This portal serves as the primary verification point for citizens, public authorities, and partner organizations to authenticate the credentials of EFPRP Pro-Bono Operatives (PBOs) in real-time.

## Key Features
- **Native Barcode Intelligence:** Optimized for field use with mobile devices to extract ID data directly from physical cards.
- **Instant Authentication:** Direct matching against the European Registry for PBO status confirmation.
- **Clearance Verification:** Real-time display of EIRT Authorization and AWID Investigator status for active officers.

## Security & GDPR Compliance
The EFPRP prioritizes data sovereignty and privacy. This system is engineered for strict compliance with European data protection standards:
- **Client-Side Processing:** All barcode scanning and image analysis occur locally on the user's device. 
- **Privacy by Design:** No photographs, biometric data, or telemetry are ever stored or transmitted to external servers.
- **Zero Tracking:** This portal operates without marketing cookies or invasive data harvesting.
